# 🤖 Haffy AI v1.0

A WhatsApp-based AI assistant with memory, personality, and future IoT integration plans.

---

## 🚀 Overview

Haffy AI is a personal AI assistant developed by Hafsal.

The goal of Haffy AI is to create an intelligent AI companion that can communicate naturally, remember users, and eventually evolve into a complete AI-powered IoT assistant capable of interacting with smart devices.

Version 1 focuses on conversational intelligence, memory, and user personalization through WhatsApp.

---

### 💬 WhatsApp Integration
